# Source Hygiene

No factual claim in a board output may be stated without an appropriate certainty level. This applies to both strategic analysis and to any statements about the archetypes themselves.

See `archetype-guidelines.md` for the full labelling framework.

## Core Rule

Every claim in a board output must be one of:
- **A stated principle** — directly derived from a documented public source.
- **An inferred position** — reasoning from known principles, clearly labelled as inference.
- **A hypothesis** — generated reasoning beyond the public record, clearly labelled.
- **A speculation** — highly uncertain option included for creative breadth, clearly flagged.

If a claim does not fit any of these categories without fabrication, it is not included in the output. A clearly named gap is always preferable to a plausible-sounding fabrication.

## Checklist Before Delivering a Board Output

- [ ] The archetype disclaimer appears at the top.
- [ ] No statement is presented as an actual view of a named individual without a public source.
- [ ] Speculative options are labelled as speculative.
- [ ] Every recommendation is labelled as Act Now / Pilot / Investigate Further / Discard.
- [ ] Unverified assumptions driving a recommendation are explicitly stated.
- [ ] Open questions that require more data before deciding are listed at the end.
